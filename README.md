## What to do  
1. Clone this repo     
2. Run `npm install` to download the node_modules folder   
3. Run `npm start`
   
## Technologies ⚙️   
 
* HTML5   
* CSS3 
* React js
* Redux
* React Redux
* React Router
* Firebase and Firestore for both authentication and handling data
* Stripe for handling payment
